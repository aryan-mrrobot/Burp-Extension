
XSS HUNTER PRO - BURP SUITE SETUP INSTRUCTIONS
================================================

STEP 1: VERIFY REQUIREMENTS
---------------------------
- Burp Suite Professional (extension support)
- Jython/Python environment configured
- All project files in same directory

STEP 2: LOAD EXTENSION
---------------------
1. Open Burp Suite Professional
2. Go to "Extender" tab
3. Click "Extensions" sub-tab
4. Click "Add" button
5. Select "Python" as Extension type
6. Browse to main.py file in this directory
7. Click "Next"

STEP 3: VERIFY INSTALLATION
--------------------------
Expected output in Extensions → Output tab:
- "All custom modules imported successfully"
- "Starting component initialization..."
- "All components initialized successfully"
- "XSS Hunter Pro Extension Loaded Successfully!"

Expected UI:
- New "XSS Hunter" tab in main Burp interface
- OR "Minimal Test" tab if debugging mode is active

STEP 4: TROUBLESHOOTING
----------------------
If extension fails to load:

A) CHECK OUTPUT TAB
   - Look for import errors
   - Verify all modules loaded successfully
   - Check for component initialization errors

B) CHECK ERRORS TAB
   - Look for Java/Python compatibility issues
   - Note specific error messages

C) COMMON ERROR FIXES

   ERROR: "TypeError: can't convert JPanel instance to java.awt.Component"
   FIX: 
   - Extension will automatically try debug versions
   - Check if "Minimal Test" or "Simple Test" tab appears
   - This indicates Java/Jython compatibility issue

   ERROR: "ModuleNotFoundError" 
   FIX:
   - Verify all files are in same directory as main.py
   - Check that ui/, utils/, csp/, payloads/ folders exist
   - Ensure __init__.py files are present in ui/ and utils/

   ERROR: "Extension fails to load"
   FIX:
   - Go to Extender → Options
   - Check Python Environment settings
   - Try reloading the extension

STEP 5: USAGE
------------
Once loaded successfully:

1. Scanner Tab:
   - Enter target URL
   - Select scan options
   - Choose payload type
   - Click "Start Scan"

2. Results Tab:
